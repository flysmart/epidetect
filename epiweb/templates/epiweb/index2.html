Project Description

EpiDetect is a software tool that supports early detection of infectious disease outbreak. Outbreak knowledge discovered by mining texts from the Social Media, with Twitter, Facebook (non-profile pages), and Google Search Engine being the primary data sources.

Available at: epidetect.com/outbreaks

Tweet

Application Features

  • Automatic Detection of Disease Outbreak
  • Location Detection of Disease Outbreak
  • Language Detection of Disease Reports
  • Geolocation and Geocoding of Locations
  • Web Compatibility
